# :clipboard: javascript-cheatsheet
Cheatsheet for Javascript Syntax and Functionalities

## Data Types
* ### Primitive
  * Boolean
  * Null
  * Undefined
  * Number
  * BigInt
  * String
  * Symbol
  
* ### Objects
  * Array
  * Typed Array
  * Map
  * Set
  * Date
  * etc.
  
### Examples:
```javascript
let myString = "string"; // String

let myNum = 123; // Number

let myBool = true; // Boolean
```
